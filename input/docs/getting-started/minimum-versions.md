
## Visual Studio Minimums

Visual Studio 2017 is the base standard. Visual Studio 2019 and beyond is recommended.

## Platform Minimums

ReactiveUI has the following minimum platform requirements:

| Platform | Minimum |
|----------|---------|
| .Net Framework | 4.6.1 |
| .Net Standard | 2.0 |
| .Net Core | 2.0 |
| UWP | 10.0.17763.0 |
| Xamarin Android | 8.1 (Oreo) |
| Xamarin iOS | 1.0 |
| Xamarin Mac | 2.0 |
| Xamarin TVOS | 1.0 |
| Tizen | 4.0 |
